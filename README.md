# 🍽️ Café Delight - Restaurant Website

A modern and responsive multi-page restaurant website built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Multi-page website (Home, Menu, About, Contact)
- Fully responsive design (mobile + desktop)
- Clean UI design
- Menu section with food items
- Contact form layout

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure
- index.html → Home Page  
- menu.html → Menu Page  
- about.html → About Page  
- contact.html → Contact Page  
- style.css → Styling  
- script.js → JavaScript
